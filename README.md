## Develop and test the Matlab code locally
Matlab code is in the src directory. Included is a matlab function to test it, `test_matlab.m`. Once that works within the Matlab IDE, the next step is to verify the command line operation with `test_matlab.sh`. Only at that point is it worth proceeding to compilation.
